## bash

`.bashrc` is normally in your home dir

## borgmatic

borgomatic config is in `~/.config/borgmatic/`.

## emacs

emacs config is in `~/.emacs.d/`.

## kitty

kitty config is in `~/.config/kitty/`.

## minecraft

Minecraft shortcut is for `~/.local/share/applications/`.

An icon should be in `/usr/share/pixmaps/`.

## nvim

It's a custom config for [Kickstart](https://github.com/nvim-lua/kickstart.nvim), so it should be placed in `~/.config/nvim/lua/custom/plugins/`.
